# Tick-Tick (локальная версия)

Минимальный клиент задач, полностью работающий в браузере. В репозитории уже есть всё, чтобы поднять рабочее локальное приложение: JS-модули, стили, календарь и состояния с демо-данными.

## Как скачать

```bash
git clone https://github.com/VITperson/tick-tick.git
cd tick-tick
```

## Запуск

1. Откройте `index.html` в браузере (двойной клик, drag/drop или `file://`); либо  
2. Запустите статический сервер (например, `npx http-server` или `python -m http.server 8080`) и откройте `http://localhost:8080`.

После загрузки приложение автоматически заполнит сутки на текущую неделю и переключится на *недельный* вид. Calendar view поддерживает создание/редактирование задач, перетаскивание, таймзоны и т.д.

## Структура

- `src/` — код приложения (модели, компоненты, виды).  
- `styles/` — базовые стили.  
- `assets/` — медиа/шаблоны.  
- `.gitignore` — исключает локальные Markdown-файлы, которые остаются документами вне репозитория.
